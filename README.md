# SlangBit

**Translate anywhere on Windows. Completely offline. Powered by local AI models.**

SlangBit is a Windows desktop application that allows you to translate selected text in virtually any application using a global hotkey.

No cloud services.

No tracking.

No subscriptions.

No text is sent outside your computer.

---

## Features

* Global hotkey translation
* Fully offline translation
* Local AI-powered language models
* Automatic package management
* One-click language installation
* System tray integration
* Windows startup support
* Clipboard replacement workflow
* No account required

---

## How It Works

1. Select text in any application
2. Copy the text
3. Press **ALT + Z**
4. SlangBit translates the content locally
5. The translated text is copied back to the clipboard
6. Paste normally

Supported applications include:

* Web browsers
* Office software
* Social media platforms
* Email clients
* Text editors
* Most Windows desktop applications

---

## Privacy First

SlangBit is designed with privacy as a core principle.

All translations are performed locally using AI models installed on your computer.

Your text is never transmitted to:

* Cloud APIs
* External translation services
* Third-party servers

No user accounts are required.

No telemetry is collected.

No personal data leaves your device.

---

## Language Packages

SlangBit uses downloadable language packages based on local AI translation models.

Packages can be installed or removed directly from the built-in Package Manager.

Current supported language pairs include:

* Italian ↔ English
* Italian ↔ Spanish
* English ↔ Spanish
* English ↔ French
* English ↔ German
* English ↔ Chinese

Additional language packages may be added in future releases.

---

## Technology

SlangBit is built using:

* Flutter
* CTranslate2
* OPUS-MT models
* Marian tokenizers

The application runs entirely on the user's machine.
